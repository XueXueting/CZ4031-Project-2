Query Annotator
The query annotator allows users to select their database schema and submit an SQL query.
If the query is valid, it will be annotated with the type of algorithms used for the operations in the Query Execution Plan.
The graphical QEP will also be shown.

Steps for running:
1) Open the project in Pycharm
2) Open the terminal in Pycharm is activate the virtual environment using .\codes\venv\Scripts\activate
3) Set the database authentication details in database.ini within the codes folder
4) Run project.py
